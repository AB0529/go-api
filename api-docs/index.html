Paths

screenshots

retrieve a screenshot

GET /screenshots/{name}

By passing in the name of the screenshot, you can search for specific screenshots by name

name: string
in path

the name of the screenshot

200 OK

search results matching criteria

type
object
404 Not Found
Screenshot404

screenshot not found

Response Content-Types: application/json
Response Example (200 OK) 
{
  "status": 200,
  "state": "ok",
  "result": {
    "img": "Binary Image Here",
    "name": "cool-cat-600",
    "mime": "image/png",
    "timestamp": "2009-11-10 23:00:00 +0000 UTC m=+0.000000000"
  }
}
Response Example (404 Not Found) 
{
  "status": 404,
  "state": "fail",
  "result": "error: could not find screenshot"
}
